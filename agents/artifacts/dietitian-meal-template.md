## Emergency / Travel Options

When normal meals aren't possible:

| Situation         | Solution                                              | Calories | Protein |
|-------------------|-------------------------------------------------------|----------|---------|
| No time breakfast | Pre-made shake (whey + oats + banana + milk)          | 550      | 35g     |
| Skipped lunch     | Double snack: shake + protein bar + nuts              | 800      | 45g     |
| Travel day        | Protein bars (2) + nuts (100g) + dried fruit          | 900      | 40g     |
| Restaurant        | Order protein + starch, add olive oil/butter request  | Variable | 30g+    |
| Hotel breakfast   | Eggs (3+) + cheese + bread + yogurt + fruit           | 600      | 35g     |

**Portable emergency kit** (keep stocked):
- 2 protein bars (30g protein total)
- 100g mixed nuts (600 kcal)
- 50g dried fruit (150 kcal)
- Single-serve seed butter packets (tahini/sunflower)
- Protein powder single-serve packets

## Substitution Rules

### Protein Swaps (equal protein, similar calories)
| If avoiding...    | Replace with...                          |
|-------------------|------------------------------------------|
| Chicken           | Turkey, pork tenderloin                  |
| Salmon            | Trout, mackerel, sardines, cod + olive oil |
| Beef              | Lamb, bison, pork                        |
| Eggs              | Egg whites (3:1) + extra fat source      |

| Oats              | Cream of rice, muesli, granola           |
| Bread             | Rice cakes + tahini, wraps               |

### Fat Swaps (similar calories, no peanut butter or nut butters)
| Standard          | Alternatives                             |
|-------------------|------------------------------------------|
| ~~Peanut butter~~ | Tahini, sunflower seed butter, coconut cream, avocado |
| ~~Nut butters~~   | Tahini, sunflower seed butter, coconut cream, avocado |

| Nuts (peanuts)    | Almonds, cashews, walnuts, macadamias    |

## Compliance Checkpoints

DIETITIAN monitors weekly:

| Metric                          | Target              | If Not Met                        |
|---------------------------------|---------------------|-----------------------------------|
| Meals eaten per day             | 5/5                 | Identify skipped slot, add shake  |
| Protein per meal                | ≥20g each           | Increase portion or add source    |
| Total daily calories (average)  | 2,400-2,650         | If low: add calorie boosters      |

## Handoff to CHEF

For each slot, DIETITIAN provides:

```json
{
  "from_agent": "DIETITIAN",
  "to_agent": "CHEF",
  "data_type": "meal_slot_spec",
  "payload": {
    "slot": "dinner",
    "day": "Monday",
    "protein_g": 28,
    "carbs_g": 65,
    "fat_g": 22,
    "calories": 620,
    "constraints": ["no_peanut_butter", "batch_cookable", "Mediterranean_cuisine"],
    "primary_protein": "chicken_thigh",
    "prep_time_max_min": 45,
    "batch_portion": 3
  }
}
```

CHEF returns complete recipe with:
- Exact ingredients with gram weights
- Step-by-step instructions
- Batch cooking modifications
- Storage and reheating instructions
- Macro breakdown verification
